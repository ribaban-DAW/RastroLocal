# RastroLocal

## Descripción

Página de prueba para la asignatura Proyecto Intermodular del Grado Superior de Desarrollo de Aplicaciones Web para probar React.

Incluye diseño de la idea principal en HTML y su paso a componentes utilizando React, añadiendo mejoras como navegación a través de la página utilizando React Router DOM y obtención de productos utilizando axios.

## Uso

Clonar el repositorio
```shell
git clone https://github.com/ribaban-DAW/RastroLocal
```

Ir a la carpeta
```shell
cd RastroLocal/RastroLocalReact
```

Instalar las dependencias
```shell
pnpm install
```

Ejecutar el servidor
```shell
pnpm dev
```

> [!NOTE]
>
> Puedes utilizar `npm` en lugar de `pnpm` si no lo tienes
